========================
Instrucciones de uso
========================
Uso Ejecutable

1- Haz doble click en ProjectoJuego.jar
2- Selecciona la dificultad del juego
3- Disfruta el juego
4- Para fnalizar solo cierra la ventana

Uso por Netbeans

1- Haz clic en “File” y seleccionas “Open Project”, buscar la carpeta ya descomprimida.
2- Una vez ubicada la carpeta, selecciónala y haz clic en el botón “Open Project” y se abrirá el proyecto.
3- Vas al packages “código”, ahí encontraras varios “código.java”.
4- Selecciona el main.java y das click a “Run Project” con el cual se ejecutará el juego.
5- Una vez terminado el juego, cierra la ventana para finalizar y cierra el programa NetBeans.
